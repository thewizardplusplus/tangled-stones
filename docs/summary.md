# Summary (EN / [RU](summary_ru.md))

[<< Back](README.md)

![](screenshot.png)

There is a set of paired connected stones on the game field. The player's task is to untangle them and place them in the hole at the bottom of the screen.

By default, the game starts with a one-stone grid. After each completed level, the side of the grid grows by one up to a maximum of 20. (However, automatic growth can be disabled in the game settings.)
